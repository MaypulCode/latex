# 周波数分割なし（No Sharing）

## 概要

比較手法として使用する、周波数分割を行わない方式。マクロセル・マイクロセルともに全帯域（273 RB）を使用する。

> [!WARNING]
> この手法は干渉対策を行わないため、同一チャネル干渉が発生する。

---

## RB 割当

| セル         | 割当 RB 数 |
| :----------- | :--------- |
| マクロセル   | 273        |
| マイクロセル | 273        |

$$N_{RB}^{macro} = N_{RB}^{micro} = 273$$

---

## 干渉モデル

### 周波数分割方式との違い

| 項目       | 周波数分割あり     | 周波数分割なし（本手法） |
| :--------- | :----------------- | :----------------------- |
| 帯域使用   | 分割（137+136 RB） | 全帯域（273 RB）         |
| 干渉タイプ | 隣接チャネル干渉   | **同一チャネル干渉**     |
| 干渉減衰   | ACIR（32.7 dB）    | **なし（0 dB）**         |

### 同一チャネル干渉（Co-channel Interference）

周波数分割を行わないため、他の基地局からの受信電力がそのまま干渉電力となる：

$$SINR = \frac{S}{I_{co} + N_0}$$

| 記号     | 説明                             |
| :------- | :------------------------------- |
| $S$      | サービングセルからの受信電力 [W] |
| $I_{co}$ | 同一チャネル干渉電力 [W]         |
| $N_0$    | 熱雑音電力 [W]                   |

$$I_{co} = P_{rx,interferer}$$

隣接チャネル干渉モデル（ACIR）では干渉が 32.7dB 減衰されるが、同一チャネル干渉では減衰なし。

---

## 特徴

| 項目           | 説明                 |
| :------------- | :------------------- |
| スペクトル効率 | 高（全帯域使用可能） |
| 干渉レベル     | 高（対策なし）       |
| 実装複雑度     | 最小                 |
| 使用場面       | ベースライン比較     |

---

## 実装

```rust
pub struct NoSharing {
    total_rbs: u32,  // 273
}

impl SpectrumSharing for NoSharing {
    fn allocate(&self, ...) -> AllocationResult {
        AllocationResult {
            macro_rbs: self.total_rbs,  // 273
            micro_rbs: self.total_rbs,  // 273
            micro_tx_power_dbm: current_power,
        }
    }
}
```

---

## 期待される結果

- 近距離（50m）: 干渉大 → スループット低下、利用率上昇
- 遠距離（200m+）: 干渉小 → Static Division と同等の性能
