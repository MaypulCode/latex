# シミュレーション設定

## モンテカルロシミュレーション

| 記号        | 説明                   | 値    |
| :---------- | :--------------------- | :---- |
| $d_{min}$   | 最小基地局間距離       | 50 m  |
| $d_{max}$   | 最大基地局間距離       | 250 m |
| $\Delta d$  | 距離間隔               | 5 m   |
| $N_d$       | 距離パターン数         | 41    |
| $N_{MC}$    | 各距離での反復回数     | 50    |
| $N_{total}$ | 総シミュレーション回数 | 2,050 |

---

## 距離パターン

$$d \in \{50, 55, 60, ..., 250\} \text{ [m]}$$

各基地局間距離に対して 50 回のシミュレーションを実行し、結果を平均化（モンテカルロシミュレーション）。

---

## 乱数シード

各シナリオは異なる乱数シード（0〜49）で生成され、以下が変化:

- UE 配置（位置座標）
- トラフィックパターン（パケット到着時刻、データ量）

---

## シミュレーション時間

| パラメータ           | 値      |
| :------------------- | :------ |
| シミュレーション時間 | 60 s    |
| スロット長           | 0.5 ms  |
| 総スロット数         | 120,000 |
