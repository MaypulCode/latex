# 時間・周波数パラメータ

## 時間パラメータ

| 記号             | 説明                           | 値      |
| :--------------- | :----------------------------- | :------ |
| $\Delta t$       | タイムステップ長（1 スロット） | 0.5 ms  |
| $\Delta f_{SCS}$ | サブキャリア間隔               | 30 kHz  |
| $T_{sim}$        | シミュレーション時間           | 60 s    |
| $N_{slots}$      | 総スロット数                   | 120,000 |

---

## 周波数パラメータ

| 記号              | 説明           | 値                  |
| :---------------- | :------------- | :------------------ |
| $f_c$             | 中心周波数     | 4.85 GHz (Sub6)     |
| $BW$              | システム帯域幅 | 100 MHz             |
| $N_{RB}^{total}$  | 総 RB 数       | 273                 |
| $N_{RB}^{shared}$ | 共用 RB 数     | 0（周波数分割方式） |

---

## リソースブロック（RB）構成

| パラメータ               | 値            |
| :----------------------- | :------------ |
| サブキャリア数/RB        | 12            |
| OFDM シンボル数/スロット | 14            |
| リソースエレメント数/RB  | 168 (12 × 14) |

---

## RB 割当制約

$$N_{RB}^{macro} + N_{RB}^{micro} = N_{RB}^{total} = 273$$

周波数分割方式により、マクロセルとマイクロセルに割り当てる RB 数の合計は常に 273。
