# 概要とスコープ

## 1. 目的

本シミュレータは、ローカル 5G 環境における**マクロセルとマイクロセル間の周波数共用**を評価するためのツールです。

### 背景

- ローカル 5G では、既存のマクロセル（非優先）とローカル 5G マイクロセル（優先）が同一周波数帯を共用
- 適切なリソース割り当てにより、両者の公平性と効率を両立させることが課題

### 目標

1. 周波数共用手法の性能評価
2. RB 割当・送信電力最適化アルゴリズムの検証
3. 評価指標の定量的比較

---

## 2. スコープ

| 項目           | 内容                                  | 備考                           |
| :------------- | :------------------------------------ | :----------------------------- |
| **対象レイヤ** | MAC 層スケジューリング                | 物理層はチャネルモデルで抽象化 |
| **環境**       | HetNet（マクロ 1 基 + マイクロ 1 基） | ローカル 5G 共用環境           |
| **手法**       | 静的分割、二段階最適化                | 比較手法と提案手法             |
| **評価指標**   | 周波数効率、公平性、エネルギー効率    | Jain's Fairness Index 使用     |

### スコープ外

- モビリティ（UE は静止）
- ハンドオーバー
- 複数マクロセル/マイクロセル

---

## 3. 用語定義

| 用語       | 定義                                                                                 |
| :--------- | :----------------------------------------------------------------------------------- |
| **gNB**    | 5G 基地局（gNodeB）                                                                  |
| **UE**     | ユーザ端末（User Equipment）                                                         |
| **RB**     | リソースブロック（Resource Block）- 周波数方向 12 サブキャリア × 時間方向 1 スロット |
| **CQI**    | チャネル品質指標（Channel Quality Indicator）- 1〜15 の整数値                        |
| **MCS**    | 変調符号化方式（Modulation and Coding Scheme）                                       |
| **SINR**   | 信号対干渉雑音比（Signal to Interference plus Noise Ratio）                          |
| **HetNet** | 異種ネットワーク（Heterogeneous Network）- マクロセルとスモールセルの混在環境        |
